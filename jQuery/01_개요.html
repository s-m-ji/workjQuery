jQuery

        jQuery는 웹  개발에서 자주 사용되는 JavaaScript 라이브러리
        HTML 문서의 조작, 이벤트 처리, 애니메이션, AJAX 요청 등을 간단하고 효율적으로 처리

jQuery 라이브러리 연결 방법

        1) 파일
        2) 링크(CDN 방식)

jQuery 실행

        window.onload = function(){
            // 실행 내용
        }

        window.onload 요소가 다 로딩되고 외부 문서들도 다 가지고 온 뒤에 실행됨.
        한번만 실행 가능하며 마지막 것만 적용됨.
        * 이벤트 핸들러를 이용하면 여러번 적용 가능
